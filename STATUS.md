# HyperTide Status

HyperTide is currently a `v0.1.0 preview` open source release.

## Available Now

- Server and CLI workflows for repo, branch, lock, sync, checkout, add, submit, status, diff, log, rollback, checkpoint, audit, and witness operations.
- PostgreSQL-backed server state with Docker Compose deployment assets.
- Content-addressable storage for large binary assets.
- MIT-licensed Community Edition source code.

## In Roadmap

- Public desktop or web GUI.
- DCC integrations for tools such as Unreal, Unity, Blender, Maya, and Houdini.
- Enterprise extensions around identity, advanced policy, managed deployment, compliance export, and SLA support.
- More release automation for signed multi-platform binaries.

The public repository is intended to be usable for the Server + CLI path first. See [ROADMAP.md](ROADMAP.md) for planned follow-up work.
