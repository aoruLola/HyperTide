# Support

HyperTide Community Edition support happens in public project channels.

## Questions

- Check [README.md](README.md), [docs/getting-started.md](docs/getting-started.md), and [docs/cli/user-guide.md](docs/cli/user-guide.md) first.
- Open a GitHub Discussion or Issue in `openLYURA/HyperTide` when the question may help other users.

## Bugs

Open a GitHub Issue with:

- HyperTide version or commit SHA.
- Operating system and shell.
- Server deployment mode.
- Exact CLI command or API call.
- Expected behavior and actual behavior.

## Security

Do not open public issues for vulnerabilities. Follow [SECURITY.md](SECURITY.md).

## Enterprise Support

Commercial support, managed deployment, and private enterprise extensions are separate from this public Community Edition repository.
